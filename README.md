# Locked-In

## Overview
Locked-In is a productivity and preparedness tool designed to support individuals who struggle with short-term memory, motivation, and daily task management. The goal of the project is to help users stay organized, remember essential responsibilities, and build consistent routines in a structured and accessible way.

In today’s fast-paced environment, many individuals—especially college students—face increasing pressure to manage multiple responsibilities at once. Locked-In aims to reduce that pressure by making daily expectations clearer, more manageable, and easier to follow through on.

---

## Problem
Many individuals struggle to consistently complete day-to-day tasks due to cognitive overload, distractions, or memory-related challenges. This is especially common among:

- Full-time college students balancing academics, work, and personal life  
- Neurodivergent individuals (such as those with ADHD or executive functioning difficulties)  
- Individuals with short-term memory limitations  

Existing tools like standard to-do lists or calendar apps often fall short because they assume users can independently manage, prioritize, and remember tasks without additional support. These tools tend to be passive rather than actively supportive, which can lead to missed responsibilities, frustration, and decreased productivity.

---

## Solution
Locked-In provides a structured and supportive system that helps users stay on track throughout their day. Instead of simply listing tasks, it reinforces task completion through reminders, guided steps, and motivational support.

The application focuses on:

- Breaking down tasks into smaller, manageable actions  
- Providing consistent and timely reminders  
- Reinforcing routines through repetition  
- Offering motivation to encourage follow-through  

The goal is to reduce mental strain while helping users build habits and maintain consistency.

---

## Intended Users
- College students managing multiple responsibilities  
- Neurodivergent individuals who benefit from structured systems  
- Individuals with short-term memory challenges  
- Anyone who struggles with staying organized or motivated  

---

## Planned Features
- Add and organize daily tasks  
- Break larger tasks into smaller steps  
- Set recurring reminders for routines  
- Provide real-time prompts for incomplete tasks  
- Track completed vs. missed tasks  
- Display daily and weekly task summaries  
- Highlight patterns in productivity and consistency  
- Optional motivational prompts or reinforcement system
